# Cultif - Recipe Discovery App

Cultif is a platform that allows chefs to monetize and share their recipes, and users to discover exciting new recipes from around the world that fit their dietary goals.

## Features

- **Onboarding**: 3-screen onboarding flow introducing the app
- **Authentication**: Sign up/Sign in using Convex Auth
- **Discover Screen**: 
  - Category-based browsing (Breakfast, Dinner, Desserts)
  - Taste by Country section
  - Taste by Diet section
  - Featured recipe highlights
  - Save recipes functionality
  - Bottom navigation
  - Floating action button

## Tech Stack

- **Frontend**: React + TypeScript + Vite
- **Backend**: Convex (Database + Serverless Functions)
- **Authentication**: Convex Auth
- **Routing**: React Router DOM

## Setup Instructions

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager
- Convex account (sign up at [convex.dev](https://convex.dev))

### Installation

1. **Clone the repository** (or navigate to the project directory)

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Initialize Convex**:
   ```bash
   npx convex dev
   ```
   
   This will:
   - Prompt you to log in to Convex (if not already logged in)
   - Create a new Convex project (or connect to existing)
   - Generate the `.env.local` file with your `VITE_CONVEX_URL`
   - Start the Convex dev server

4. **Set up Convex Auth**:
   ```bash
   npx @convex-dev/auth
   ```
   
   Follow the prompts to configure authentication providers (OAuth, Magic Links, or Password).

5. **Start the development server**:
   
   In one terminal:
   ```bash
   npm run dev:convex
   ```
   
   In another terminal:
   ```bash
   npm run dev:vite
   ```
   
   Or use the combined command (may require process manager):
   ```bash
   npm run dev
   ```

6. **Open the app**:
   
   Navigate to `http://localhost:5173` (or the port shown in the terminal)
