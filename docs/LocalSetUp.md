Quick Start
If you've already set up the project, you can run:
npm run dev
This will start both the Convex backend and the Vite development server simultaneously.
Separate Commands (Recommended)
For better control, run these in two separate terminals:
Terminal 1 - Start Convex backend:
npm run dev:convex
Terminal 2 - Start Vite dev server:
npm run dev:vite
Then open your browser to http://localhost:5173 (or the port shown in the terminal).
